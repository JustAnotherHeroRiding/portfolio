# Portfolio design rules

- Never style interface copy as all-uppercase and never use CSS text transforms that force uppercase.
- Technology chips use the established logo-and-label treatment with a Nord surface, visible border, and fully rounded pill shape.
- Keep controls optically centered with deliberate spacing; use SVG icons instead of text glyphs when font metrics can misalign them.
- The project cabinet uses clean Nordic/Dutch architectural forms: modular composition, restrained asymmetry, pale structural surfaces, graphite framing, and Nord accents.
- Project artifacts use a consistent Braun/Apple-inspired product language: simple recognizable silhouettes, soft bevels, matte off-white and graphite materials, one functional accent color, and restrained motion.
- Prefer stylized clarity over realism, ornamental detail, neon effects, or placeholder-like primitive assemblies.
